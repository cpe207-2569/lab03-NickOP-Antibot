Favorites

สามารถคลิกที่ชื่อเว็บไซต์ เพื่อเข้าเว็บไชต์ได้

Youtube

แพลตฟอร์มเว็บไซต์และแอปพลิเคชันสำหรับรับชม ฟังเพลง และแชร์วิดีโอออนไลน์ฟรีจากทั่วโลก โดยผู้ใช้งานสามารถเปิดช่องของตัวเองเพื่ออัปโหลดคลิปวิดีโอ ถ่ายทอดสด และสร้างรายได้จากยอดผู้ชมหรือโฆษณา

 
Github

เว็บไซต์คลาวด์สำหรับนักพัฒนาที่ github.com ใช้จัดเก็บ แชร์ และทำงานร่วมกันในโปรเจกต์เขียนโค้ด โดยทำงานร่วมกับระบบ Git เพื่อบันทึกประวัติการแก้ไข ช่วยให้คนในทีมสามารถเขียนโค้ดพร้อมกันได้โดยไม่ทับกัน เปรียบเสมือนสังคมออนไลน์และคลังเก็บผลงานของโปรแกรมเมอร์ทั่วโลก

 
Netflix

เว็บไชต์บริการสตรีมมิงที่ให้คุณดูภาพยนตร์ ซีรีส์ และสารคดีผ่านอินเทอร์เน็ตได้แบบเหมาจ่ายรายเดือน คล้ายกับการเช่าหนังมาดูที่บ้านโดยเลือกรับชมผ่านอุปกรณ์ต่างๆ เช่น สมาร์ตทีวี คอมพิวเตอร์ หรือสมาร์ตโฟนได้ตลอดเวลาตามต้องการ

 
Steam

แพลตฟอร์มเว็บไซต์ร้านค้าและจัดจำหน่ายเกมดิจิทัลบนคอมพิวเตอร์ที่ใหญ่ที่สุดในโลก ซึ่งพัฒนาโดยบริษัท Valve ทำหน้าที่เป็นสื่อกลางให้ผู้เล่นสามารถเลือกซื้อ ดาวน์โหลด และอัปเดตเกมต่าง ๆ ได้โดยตรงผ่านระบบออนไลน์อย่างสะดวกสบาย อีกทั้งยังมีระบบสังคมออนไลน์ที่ช่วยให้ผู้ใช้งานสามารถพูดคุยกับเพื่อน หาห้องเล่นเกมด้วยกัน จัดเก็บข้อมูลความคืบหน้าของเกมไว้บนคลาวด์ ตลอดจนเป็นพื้นที่สำหรับซื้อขายไอเทมสะสมเสมือนจริงภายในเกมอีกด้วย
